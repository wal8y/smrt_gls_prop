This is the Abstract of a SmartGlasses that's Powered by AI

The Glasses

A pair of glasses that are used to see the world. take frames using the camera. and show a text on the screen. the text and the frames are processed by the chip.

The Chip

The chip is a Raspberry Pi Pico W. it is used to process the frames sent from the glasses and send them in POST request to the server

and it will get the result as a response from the server then send it to be represented on the glasses.

The Server

The server is a Ubuntu_22 Backend.

It has One API end that recives the images and process them using Local AI models in Python and then it sends the text of the processed images as a respond
